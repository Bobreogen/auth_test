/get-tokens - GET запрос, аргументы: GUID, назначение - получение access и refresh токенов

/refresh - POST запрос, аргументы: refresh токен, назначение - обновление access и refresh токенов

/delete-token - DELETE запрос, аргументы: refresh токен, назначение - удаление refresh токена

/delete-tokens - DELETE запрос, аргументы: GUID, назначение - удаление refresh токенов по GUID
